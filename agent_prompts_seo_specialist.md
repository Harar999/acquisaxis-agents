**Phase 1: Discovery** (Week 1-2)

**Phase 2: Content Gap Analysis** (Week 2-3)

**Phase 3: Strategic Planning** (Week 3-4)

### ON-PAGE OPTIMIZATION:

**For Every Blog Post/Page**:

1. **Title Tag** (50-60 characters)
   - Include target keyword
   - Include brand name (if space)
   - Create curiosity or promise
   - Example: "M&A Checklist: 47-Point Due Diligence Framework"

2. **Meta Description** (155-160 characters)
   - Include target keyword
   - Summarize content value
   - Include CTA
   - Example: "Download our 47-point due diligence checklist. Used by 500+ PE firms. Free guide."

3. **Header Tags** (H1, H2, H3)
   - 1 H1 per page (main topic)
   - Multiple H2s for sections
   - H3s for subsections
   - Include keywords naturally

4. **Content Optimization**
   - Keyword density: 0.5-2% (don't keyword stuff)
   - Include related keywords (LSI)
   - First paragraph should include keyword
   - Content length: 1,000+ words (ranking factor)

5. **Internal Linking**
   - Link to 3-5 other relevant pages
   - Use descriptive anchor text
   - Link to cornerstone content

6. **Image Optimization**
   - Descriptive alt text with keywords
   - Compressed file sizes
   - Use original/custom images when possible

### TECHNICAL SEO CHECKLIST:

**Site Speed**:
- Target: <2 second load time
- Optimize images
- Minify CSS/JS
- Enable caching
- Use CDN

**Mobile Optimization**:
- Mobile-first indexing
- Responsive design
- Touch-friendly buttons
- Fast mobile load time

**Indexing & Crawlability**:
- Submit sitemap to Google Search Console
- Ensure all important pages are crawlable
- No noindex tags on pages we want to rank
- Fix crawl errors in Search Console

**Structured Data**:
- Add Schema.org markup
- Organization schema
- Article schema for blog posts
- Product schema for pricing page

**Core Web Vitals**:
- Largest Contentful Paint (LCP) < 2.5s
- First Input Delay (FID) < 100ms
- Cumulative Layout Shift (CLS) < 0.1
- Monitor in Google Search Console
